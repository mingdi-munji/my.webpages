css3 스타일 시트로 작성하는 웹페이지

스타일 시트의 특징 
        - head태그 내에서만 작성가능
        - 대괄호{}를 사용한 같은 태그들에 같은 설정이 적용된다.
        - style태그는 여러번 작성이 가능하며, 스타일 시트들이 합쳐서 적용된다.
        - 셀렉터 내의 내용은 ;(세미콜론)으로 마무리를 시켜주어야한다.

css3 스타일 시트를 작성하는 방법

  • 바탕색 설정하기 : 대괄호{}사이에 background-color: 색깔 ;으로 설정할 수 있다 (웹페이지 전체 바탕색을 설정)
  • 글자의 색깔 설정하기 : 대괄호{}사이에 폰트 ex.)h3등으로 설정한 뒤, color: 색깔; 로 설정한다
  • hr의 색깔 (나누는 선) 을 바꾸는 법 : 대괄호{}사이에 border: -px 색깔 ; 로 설정한다
  • 단어(span)의 색깔바꾸는 법 : 대괄호{}사이에 color: 색깔 front-size : - px ; 으로 설정한다.
  1. style 사이에 원하는 스타일 시트를 작성
  2. style 속성에 스타일 시트 작성
  3. 스타일 시트를 작성하고 link 태그나 @import로 불러오기

기타 관련 내용들 내용들을 주석으로 달때에는 <>안에 /*내용*/의 방법으로 내용에 필요한 주석물을 작성할 수 있다.
<> 내에 스타일속성(style ="")을 적용하는 경우 작성한 경우에서만 적용되고 다른 태그는 적용되지않는다.

css로 웹페이지 꾸미기 - 코드관련

-->>css를 활용하여 웹페이지의 스타일을 적용하는 법에 대한 설명

  • css를 위한 섹션의 기본적인 구조
  • section -> article -> form
  • label id => '내용을 구별하기 위해 사용'
  • *하나의 section안에 여러가지 article로 구성된다.

셀렉터

셀렉터의 종류 = "자식셀렉터", "자손 셀렉터", "가상셀렉터"
  • 모든 셀렉터
    • - "*"을 사용하여 나타낸다.
  • 자식 셀렉터
    • - 자식셀렉터는 ">"로 나타낸다. (부모>자식)
  • 자손 셀렉터
    • - 자손셀렉터는 ","으로 나타낸다. (A,A의 자손)
  • 가상 셀렉터 - 일정 상황에만 적용된다.
    <가상 셀렉터의 종류>
    마우스 :hover :active
    폼요소 :focus
    링크 :link :visited
    블록 :first-letter :first-line
    구조 :nth-child(even) :nth-child(1)
색과 텍스트 꾸미기
  • 색 (r,g,b)
    • - 색을 꾸미는 것은 3가지가 존재한다
    1. color:
      2. -> 글자의 색을 변경한다.
    2. background-color:
      3. -> 배경색을 변경한다.
    3. border-color:
      4. -> 테두리의 색을 변경한다.
  • 텍스트의 출력형태
    • text-indent: 
     <>안에 legnth 
     <>안에 precentage
    text-align: leftright centerjustify
    text-decoration: noneunderline overlineline-through;

    **예시**

    text-indent:3em

    text-aligh:center

    none

    overline

    underline

    line-through

  • 폰트
    • - serif(serif있는 서채) / Arial(serif아닌 서채) / consolas(글자폭 동일)

    Serif "Hello" -(위쪽모서리 o)

    sans-serif "Hello" -(위쪽모서리 x)

    monospace "Hello" -(글자 폭 동일)

    **폰트이름에 공백이 존재하는 경우 ""으로 묶어서 표현한다.
