  • 자바스크립트 언어

    • - java script 언어: 조각난 소스형태로 html에 내장되어 소스파일이 컴파일 과정없이 인터브리터로 인해 바로 실행된다.(c언어의 단순화) 
                       ==> 자바스크립트의 script 태그는 head나 body내의 어디든지 들어갈 수 있다. 
                       또한 여러번의 작성이 가능하다.

    자바스크립트의 역할

    • 사용자의 입력 및 계산
    • 웹 페이지 내용 및 모양의 동작제어
    • 브라우저 제어
    • 웹 서버와의 통신
    • 웹 애플리케이션 작성

    자바스크립트 코드의 위치

    • html 태그의 이벤트 리스너 속성에 작성
    • script 태그로 작성
    • 자바스크립트 파일에 생성
    • URL부분에 작성

    이벤트

    • 이벤트 리스너의 종류
      • onmouseover - 마우스를 올렸을때
      • onmouseout - 마우스를 올리지 않았을 때
      • click - 눌렀을 때
      • onclick - 누른 뒤
      • change - 변화구
      • mousemove - 마우스가 움직일 때
      • 이벤트 리스너의 예시
        • onmouse
